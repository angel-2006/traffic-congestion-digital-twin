# Project Scope

## Project Title
Traffic Congestion Prediction using Digital Twin

## Objective
To build a traffic congestion prediction system using:
- TomTom live traffic data
- SUMO traffic simulation
- TraCI-based digital twin control
- machine learning for congestion prediction

## Study Area
Silk Board – Electronic City Corridor, Bangalore

## Main Components
1. Live traffic data collection
2. Time-series traffic dataset generation
3. SUMO-based traffic simulation
4. Digital twin calibration
5. Congestion prediction model
